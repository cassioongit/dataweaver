# Site Design Notes

This file is archived. The canonical visual system lives in [DESIGN.md](/Users/cassiomachado/Documents/Development/dataweaver/DESIGN.md).

## Purpose

Provide a lightweight implementation reference for the current UI direction without introducing a second design language.

## Summary

- **Primary color**: blue (`#0061FF`)
- **Background**: soft zinc gray (`#F1F3F5`)
- **Surface**: white
- **Typography**: Inter
- **Shape**: rounded, friendly, operational

## Layout Notes

- Fixed 88px white sidebar
- Blue active navigation state
- Full-bleed main content area
- Simple blue brand header
- Floating centered login card
- Subtle shadows only

## Component Notes

- Inputs should be solid-filled, not outlined-heavy
- Tables should prioritize scanability
- Badges should stay soft and readable
- Primary actions should use the blue brand color

## Compatibility Note

This file should not be used as the source of truth for new work.
